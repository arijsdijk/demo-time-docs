GitHub Copilot actions allow you to interact with GitHub Copilot directly from your Demo Time steps. You can open GitHub Copilot chat, start new chats, ask questions, and moreâ€”all as part of your scripted demo.

<Aside type="note" title="Samples">
  You can find Copilot actions samples on GitHub:
  <a href="https://github.com/estruyf/vscode-demo-time/tree/main/.demo">.demo/copilot-actions.json</a>

## Open Copilot chat

To open the Copilot chat panel, use the `openCopilotChat` action.

```json
{
  "action": "openCopilotChat"
}
```

## Start a new Copilot chat

To start a new Copilot chat session, use the `newCopilotChat` action.

```json
{
  "action": "newCopilotChat"
}
```

## Ask a question in Copilot chat

To ask a question, use the `askCopilotChat` action with a `message` property.

```json
{
  "action": "askCopilotChat",
  "message": "Write a function that returns 'Hello from Demo Time'."
}
```

## Start an edit chat with Copilot

To start an edit chat, use the `editCopilotChat` action with a `message` property.

```json
{
  "action": "editCopilotChat",
  "message": "Write a function that returns 'Hello from Demo Time'."
}
```

## Start an agent chat with Copilot

To start an agent chat, use the `agentCopilotChat` action with a `message` property.

```json
{
  "action": "agentCopilotChat",
  "message": "Write a function that returns 'Hello from Demo Time'."
}
```

## Close Copilot chat

To close the Copilot chat panel, use the `copilotClose` action.

```json
{
  "action": "copilotClose"
}
```
